# 📖 סדר היום של נעמי

אתר סטטי קטן עם סדר יום מפורט לתינוקת בחודש חצי שנה, עבור סבתות, סבא וכל מטפל/ת.

## 🌟 תכונות

- **סדר יום שלם** - בוקר, צהריים, וערב עם זמנים ותפריטים
- **טיפים ותעצות** - מומחיות הורים בצורה נגישה
- **רשימת קניות** - כל מה שצריך לכדי שהכל לפעול בנעימים
- **עברית RTL** - תמיכה מלאה בעברית

## 🚀 שימוש

פתח ב-browser:
```bash
open index.html
```

או בקובץ המחשב:
```
naomi-schedule/index.html
```
